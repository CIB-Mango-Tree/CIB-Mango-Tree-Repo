About
Welcome to the CIB Mango Tree, a collaborative and open-source project to compile software that tests for coordinated inauthentic behavior (CIB) in datasets of online activity. A project of Civic Tech DC, our mission is to share methods to uncover how disruptive actors seek to hack our legitimate online discourse regarding health, politics, and society.

Data Scraping

The CIB Mango Tree does not provide software for scraping social media activity. It yields standardized tests to analyze CIB in data that researchers have already scraped. Data can be scraped manually from websites using custom-built scrapers, or via APIs that correspond to major social media platforms. Some skill is required to scrape and clean data well. The CIB Mango Tree provides generic tests for CIB. Generic tests can be applied on datasets of online activity from a variety of social media and news sites. CIB Mango Tree tests are optimized for data housed in CSV files, and assumes that CIB can be detected by analyzing a variety of parameters that datasets of online activity will likely have.
Parameters include:
  • Unique Post Number (UPN) = a unique number applied to every scraped post/comment
  • Unique Post Username = the unique username of the account that made the post/comment
  • Post Time/Date = the time and date that an individual post/comment was made
  • Post Content = the written content of an individual post/comment
CIB Mango Tree tests will use some or all of the available parameters to search for signs of coordinated inauthentic behavior.
